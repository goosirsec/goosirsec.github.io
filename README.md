# 反洗钱演示系统｜企业浏览器安全能力测试靶场

这是一个部署在 GitHub Pages 上的纯静态测试页面，用于验证企业浏览器的内容识别、数据防泄漏、下载限制、权限控制和审计能力。

## 重要边界

- 页面内置记录由程序合成，不来源于真实 KYC；姓名和地址明确使用测试命名。格式化标识符仍可能与公开编号空间重叠，因此不得联系、支付、拨打、核验或用于任何生产活动。
- 内置身份证测试向量使用不存在的 `990000` 行政区码，但按 GB 11643 校验位算法生成；可测试仅检查长度/字符/校验位的规则，执行严格行政区校验的规则可能不命中。
- 内置手机号为不可拨打的 `000-TEST-xxxx` 模式。页面不声称任何真实手机号段属于保留测试段；需要真实号段格式的规则，必须在受控环境使用组织批准的测试向量。
- 银行卡使用常见支付行业测试号 `4111111111111111`，IBAN 使用公开示例格式；它们仅用于检测测试，禁止支付或账户核验。
- 没有后端、数据库、登录服务或数据上报接口。
- KYC 测试记录只保存在当前浏览器的版本化 localStorage key：`aml-browser-lab:kyc:v3`；旧版本数据不会被继续复用。
- 若企业策略禁用 localStorage/sessionStorage，页面自动降级为“内存模式”，CRUD 与日志仍可测试，但刷新即丢失；页头会持续提示该状态。
- 新增或编辑前必须确认仅使用合成测试信息。用户输入内容由页面标记为“用户确认的测试数据”，页面不验证或保证其真实性。
- 粘贴区内容不持久化；上传/拖放功能只查看浏览器提供的文件元数据，不读取文件内容。
- 摄像头和麦克风一旦授权会立即停止全部媒体轨道，不预览、不录制、不上传。
- 媒体轨道清理位于 `finally`，即授权后的状态处理异常也会尝试停止全部轨道。
- 地理位置测试只显示允许/拒绝结果，不显示或保存坐标。
- 页面不读取剪贴板，只能在用户单击后写入测试内容。默认复制脱敏姓名和敏感字段；未脱敏复制需要二次确认及确认词，完成后应立即覆盖剪贴板。
- CSV 导出会对以 `= + - @` 开头（含前导空白）的单元格加安全前缀，降低电子表格公式注入风险。
- 会话日志只记录测试时间、测试类型和结果，不记录原始 PII、坐标或文件名。
- 页面使用纯静态 CSP：只加载同源脚本、样式和字体，禁止网络连接及嵌入内容；仅为本机生成的图片/下载保留 `data:` 与 `blob:` 图片、媒体能力，并设置 `no-referrer`。
- 同源测试弹窗使用随机窗口名并在写入内容前清除 `opener`，避免窗口复用和反向控制父页面。
- 截图、水印、页面另存为、开发者工具和浏览器管理后台审计结果无法由网页可信检测，必须人工验证。

## 包含的测试面

1. 40 条结构化内置 MOCK KYC 记录，支持搜索、风险筛选、分页、选择、详情、新增、编辑、软删除、撤销与重置。
2. 敏感字段默认脱敏，用户可单击临时显示 30 秒。
3. 姓名及敏感字段脱敏的 CSV、JSON、TXT、HTML、XML、PDF、Canvas PNG 下载。
4. 未脱敏 CSV 高风险导出，展示摘要并要求输入确认词。
5. 默认脱敏的单字段、组合 KYC、JSON、Base64 剪贴板写入；确认词保护的未脱敏复制；剪贴板覆盖按钮及手动粘贴区。
6. 文件选择/拖放元数据、打印、新窗口、外部链接和普通 Blob 下载。
7. 通知、位置、摄像头、麦克风、剪贴板写入、弹窗、全屏权限测试。
8. 人工验证清单和无 PII 会话日志导出。

完整的控制点、触发方式和网页能力边界见 [TEST_MATRIX.md](./TEST_MATRIX.md)。

## 本地运行

该页面不需要安装依赖或构建：

```bash
python3 -m http.server 8080
```

然后访问 `http://127.0.0.1:8080/`。通知、媒体、地理位置、剪贴板和全屏等 API 受浏览器安全上下文、用户手势及企业策略影响；GitHub Pages 的 HTTPS 环境更接近正式测试条件。

## 发布

仓库根目录就是可发布内容，GitHub Pages 可直接从默认分支根目录提供服务。发布前至少执行：

```bash
node --check assets/app.js
node tests/security.test.mjs
node tests/dom-contract.test.mjs
```

## 测试安全提示

不要把真实个人信息粘贴、录入或替换到此测试站点，也不要拨打、联系、支付或核验任何测试向量。若规则必须匹配真实行政区、手机号段或账户格式，请只在受控环境使用组织批准的测试标识符和隔离测试账号。
